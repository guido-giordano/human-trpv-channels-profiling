# TRPV Channels — Tissue Expression Analysis (GTEx)

Structured Python workflow for exploring tissue-specific expression of TRPV channels in human transcriptomics data.

## Overview
This repository contains analysis code developed in collaboration with the Neuberger Lab (LMU Munich) in the context of a review on TRPV channels and their pharmacological relevance.

The project uses publicly available transcriptomics data from the GTEx resource to examine the expression patterns of TRPV channel genes across human tissues. TRPV channels play an important role in sensory biology, particularly in pain perception.

## Features
- Retrieval and processing of GTEx transcriptomics data  
- Extraction of TRPV gene family expression (e.g. TRPV1–TRPV6)  
- Tissue-specific expression profiling across the human transcriptome  
- Generation of summary plots and visualisations  

## Technical Highlights
- Developed in Python (pandas, numpy, matplotlib, seaborn)  
- Focused workflow for exploratory transcriptomics analysis  
- Designed for clarity and reproducibility  
- Compatible with standard Linux-based environments  

## Scope of the repository
- Analysis of TRPV channel expression across multiple human tissues  
- Integration of public transcriptomics datasets  
- Support for ongoing work on TRPV channel biology and pharmacology
